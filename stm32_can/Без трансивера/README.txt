Измеритель тока/напряжения батареи с CAN-интерфейсом.

MCU: STM32F103C8T6; 
CAN ID: 0x293, 500 кбит/с, период 100 мс;
Значения тока и напряжения имитируются;
CAN-пакет дублируется в UART для проверки.

!!!Генерируем фейковые токи так как отсутствует CAN трансивер mcp2551!!!

Обработчик таймера(каждые 100 мс);
вместо реальных усреднений с АЦП
подставляем синтезированные значения;
напряжение: 370В + небольшая синусоида для реализма;
ток: 50А при разряде, -30А при заряде (меняется по счётчику).

ИНСТРУКЦИЯ:
Создайте новый проект, в окне выбора чипа выберите STM32F103C8T6, дайте название проекту и нажмите Finish.

Настройка периферии в CubeMX1.
Слева панель Pinout & Configuration.
1. System Core->RCC
High Speed Clock: выберите Crystal/Ceramic Resonator

2. System Core->SYS
Debug: выберите Serial Wire

3. Connectivity->USART1
Mode: выберите Asynchronous
Всё остальное оставьте по умолчанию (115200, 8, N, 1)

4. Connectivity->CAN
Активируйте (Mode: Normal)
Всё остальное оставьте по умолчанию (500 кбит/с)

5. Timers->TIM2
Clock Source: Internal Clock
Prescaler: 7199
Counter Period: 9999
Перейдите на вкладку NVIC Settings йи поставьте галочку TIM2 global interrupt

6. Настройка пина PC13:
На изображении чипа найдите пин PC13.
Нажмите на него и выберите GPIO_Output.

7. Вкладка Clock Configuration:
Убедитесь, что HCLK = 72 MHz.
Если нет — нажмите на поле HCLK и выберите 72.

8. Сохраните и сгенерируйте код:
Нажмите кнопку Save.
Нажмите Project->Generate Code.

9. Замените сгенерированный код на код из файла main.c.
10. Нажмите Project->Build All. Дождитесь успешной сборки.
11. Подключите ST-Link V2 к плате (3.3V, GND, SWDIO, SWCLK).
12. Нажмите Run->Debug As->STM32 Cortex-M C/C++ Application.
13. В открывшемся окне на вкладке Debugger убедитесь, что выбран ST-LINK (OpenOCD) и интерфейс SWD.
14. Нажмите Debug. Прошивка зальётся.
15. Нажмите F8 для запуска программы.
16. Подключите USB-UART адаптер к пинам PA9 (TX) и GND вашей платы.
17. Откройте терминал (Putty, TeraTerm), выберите COM-порт, скорость 115200.
18. В терминале должны появиться строки каждые 100 мс:
ID: 0x293 | I: 50.0A | V: 375.0V | P: 18.75kW | Status: 0x00 | CRC: 0x3

Синий светодиод PC13 будет гореть.
