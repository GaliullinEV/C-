Инстукция по Linux Kernel

"Модуль ядра linux на языке C, который создаёт виртуальный сетевой интерфейс и 
посредством procfs модуль позволяет задать IPv4 адрес интерфейса, который отвечает на ping запросы."

Перед началом убедитесь, что установлены заголовки ядра и инструменты сборки.

1) Откройте терминал и перейдите в папку с исходными файлами(mainNet.c, Makefile): 
cd ~/"name"/

2) Выполните команду сборки:
make

В результате будет создан файл модуля mainNet.ko.

3) Загрузка модуля
Для активации модуля в ядре выполните:
sudo insmod mainNet.ko

4) Поднимите интерфейс:
sudo ip link set vnet0 up

5) Установите IP-адрес через procfs (этот адрес будет привязан):
echo "192.168.1.1" | sudo tee /proc/vnet_ip

6) Назначьте IP-адрес интерфейсу в системе:
sudo ip addr add 192.168.1.1/24 dev vnet0

7) Тестирование
Запустите пинг:
ping 192.168.1.1 -c 4

8) Отслеживайте логи ядра чтобы убедиться в обработке пакетов:
sudo dmesg -w | grep vnet

9) Для удаления модуля выполните:
sudo rmmod mainNet
